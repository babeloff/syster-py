= syster-py
:description: Python bindings for the syster-base SysML v2 / KerML parser

Python bindings for *syster-base* — a fast, lossless parser and semantic analyser for https://www.omg.org/spec/SysML/2.0[SysML v2] and https://www.omg.org/spec/KerML/1.0[KerML] built in Rust.

[source,python]
----
import syster_py

sf = syster_py.parse_sysml("package VehicleModel { part def Wheel; }")
for sym in sf.symbols(file_id=1):
    print(sym.kind, sym.name)
# Package  VehicleModel
# Part def Wheel
----
